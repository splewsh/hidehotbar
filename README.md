## What does Hide Hotbar do?

Hide Hotbar allows you to hide the hotbar in any F5 mode using Mod Menu's settings:

<img src="https://cdn.modrinth.com/data/cached_images/6c1c925a962a38ec94dbdbd1614a5840ceb5b2e7_0.webp" alt="Hide Hotbar Settings" width="540">

## Why?

I made this pack for [my own YouTube videos](https://www.youtube.com/@splewsh), and I thought it might be useful for other content creators or people who just prefer not to have the hotbar visible in any F5 mode.

## How to install

Use the [Fabric Loader](https://fabricmc.net/use/installer/), preferably with the [Modrinth App](https://modrinth.com/app) to keep up to date with the latest version of the pack and to easily install the dependencies. If you don't like the Modrinth App, you can use [any other Fabric-compatible launcher](https://mc-launcher.tayou.org/).

### Dependencies

Hide Hotbar requires the [Fabric API](https://modrinth.com/mod/fabric-api), [Cloth Config API](https://modrinth.com/mod/cloth-config) and is basically useless without [Mod Menu](https://modrinth.com/mod/modmenu) for configuration.
